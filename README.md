# HUMA-USV-MISSION-COMPUTER
Hüma USV İnsansız Deniz Aracı için tasarlanmış görev bilgisayarıdır
Teknofest 2025 İnsansız Deniz Aracı yarışması için tasarlamış olduğumuz ürün aynı zamanda bitirme projemdir. Ürünün öne çıkan özellikleri şu şekildedir:
-Elektromekanik birimlerden tamamen izole güç yönetimi
-Gömülü GPS
-Gömülü ve harici sensör bileşenleri
-Yedekli fonksiyonel kullanım
